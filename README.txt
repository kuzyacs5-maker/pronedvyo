# ПРО НЕДВИЖИМОСТЬ — Чёрно-белый лендинг

## Как посмотреть демо
- Откройте файл `index.html` двойным кликом — сайт откроется в браузере.

## Быстрая публикация на Netlify
1. Зайдите на https://app.netlify.com/drop
2. Перетащите в окно папку всего проекта или `index.html` + папку `assets`.
3. Сразу получите ссылку вида `https://...netlify.app`
